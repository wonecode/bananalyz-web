## What is Bananalyz?

Bananalyz is a Discord bot that lets your server predict League of Legends esports match outcomes.
Pick the winner, predict the exact score, earn points, and compete on your server leaderboard.

## Key features

- **Interactive predictions** — submit picks directly via Discord buttons on match embeds
- **Auto-sync** — matches and scores update every 5 minutes automatically
- **Leaderboards** — server-level and global rankings
- **Stats** — win rate, perfect predictions, total points via `/predictions`
- **Multi-league support** — LEC enabled now, more leagues coming

## Quick start

1. [Add Bananalyz to your server](https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=139586750592&scope=bot+applications.commands)
2. Run `/config` to set your predictions channel
3. Wait for match embeds to appear — or use `/matches` to see upcoming ones
4. Click the prediction buttons on a match embed to submit your pick
