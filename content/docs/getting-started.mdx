import { Step, Steps } from 'fumadocs-ui/components/steps';
import { Callout } from 'fumadocs-ui/components/callout';

<Steps>
  <Step>
    ### Add the bot

    [Click here to invite Bananalyz](https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=139586750592&scope=bot+applications.commands) to your server.

    Select the server you want to add it to and approve the requested permissions.

    | Permission | Why |
    |---|---|
    | Send Messages | Post match embeds and prediction results |
    | Embed Links | Display rich embeds |
    | Use Application Commands | Register slash commands |
    | Read Message History | Update existing match embeds |
  </Step>

  <Step>
    ### Configure your predictions channel

    Run `/config` in any channel (requires **Manage Server** permission):

    ```
    /config predictions-channel: #predictions
    ```

    This sets the channel where upcoming match embeds will be posted automatically.
  </Step>

  <Step>
    ### Enable leagues

    After running `/config`, a dropdown will appear letting you select which leagues to enable for your server.

    <Callout type="info">
      **LEC is enabled by default.** More leagues are coming soon.
    </Callout>
  </Step>

  <Step>
    ### Start predicting

    Once a match embed appears in your predictions channel, members can click the buttons to submit their pick:

    - Choose the **winning team**
    - Predict the **exact series score** (e.g. 3-0, 3-1, 3-2)

    Predictions lock automatically when the match starts.
  </Step>
</Steps>
