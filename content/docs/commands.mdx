## Predictions

Command: `/predictions` — show your stats and recent prediction results.

| Option | Type | Required | Description |
|---|---|---|---|
| `user` | User | No | View another user's predictions (upcoming picks are hidden) |
| `share` | Boolean | No | Post the result publicly in the channel (default: ephemeral) |

<Tabs items={['Basic', 'View someone', 'Share publicly']}>
  <Tab value="Basic">
    ```
    /predictions
    ```
  </Tab>
  <Tab value="View someone">
    ```
    /predictions user:@Singe
    ```
  </Tab>
  <Tab value="Share publicly">
    ```
    /predictions share:True
    ```
  </Tab>
</Tabs>

## Matches

Command: `/matches` — list upcoming matches that are open for predictions on your server.

No options — shows all matches with active prediction windows.

```
/matches
```

---

## Leaderboard

Command: `/leaderboard` — display the prediction leaderboard.

| Option | Type | Required | Description |
|---|---|---|---|
| `scope` | String | No | `server` (default) or `global` |

<Tabs items={['Server', 'Global']}>
  <Tab value="Server">
    ```
    /leaderboard
    ```
  </Tab>
  <Tab value="Global">
    ```
    /leaderboard scope:global
    ```
  </Tab>
</Tabs>

## Config

Command: `/config` — configure Bananalyz for your server.

<Callout type="warn" title="Admin only">
  Requires the **Manage Server** permission.
</Callout>

| Option | Type | Required | Description |
|---|---|---|---|
| `predictions-channel` | Channel | Yes | Text channel where match embeds will be posted |

After running `/config`, a league selector dropdown appears to choose which leagues to enable.

```
/config predictions-channel: #predictions
```

---

## Help

Command: `/help` — display a summary of all commands and prediction rules, with clickable command links.

```
/help
```
